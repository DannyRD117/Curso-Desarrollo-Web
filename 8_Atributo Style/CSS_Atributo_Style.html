Problemas

El gran problema de usar el atributo style en los elementos de la pagina web es que debemos copiar las propiedades a cada uno de los elementos para que tengan el mismo estilo desead Además, usar el atributo style es la peor opción para usar CSS a la presetacion de una página web HTML, debido a que, se tiene un acoplamiento máximo.
