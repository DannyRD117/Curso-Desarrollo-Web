CCS Etiqueta Style

Uso de CSS en HTML con la Etiqueta Style

En esta página web a diferencia del ejercicio anterior se usa la etiqueta style para definir las propiedades de presentación de una pagina web HTML.

Ventajas

Anque esta no es la mejor opción para implementar CSS en una página web, se tiene un menor acoplamiento entre CSS y HTML usando la etiqueta style a diferencia de usar el atributo style que tienia un acoplamiento máximo.
